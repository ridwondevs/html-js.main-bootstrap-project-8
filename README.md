# 📊 HTML JS Main Bootstrap Project 8

A simple and responsive **Student Result Checker** web application built using **HTML, Bootstrap 5, and JavaScript**. Users can enter a roll number (or ID) and instantly check their result through a clean and user-friendly interface.

## ⚙️ How It Works

1. Enter your Roll Number or Student ID.
2. Click the **Check Data** button.
3. JavaScript searches the available data.
4. If the record exists, the student's result is displayed.
5. If no record is found, an error message is shown.
